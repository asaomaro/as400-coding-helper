The Start PC Organizer (STRPCO) command starts the PC Organizer on the host system.

Note: Do not precede an entry with an asterisk unless that entry is a "special value" that is shown (on the display itself or in the help information) with an asterisk.

Text Assist (PCTA)

Specifies whether you are going to use the Personal Computer Text Assist.

The possible values are:

*YES
The Personal Computer Text Assist is going to be used.
*NO
The Personal Computer Text Assist is not going to be used.
