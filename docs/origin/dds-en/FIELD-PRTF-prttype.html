Type of name or specification for printer files (position 17)

Last Updated: 2026-07-13

The value in this position identifies the type of name in positions 19 through 28.

The valid entries for printer files are:
Entry
Meaning
R
 Record format name
Blank
 Field name

The printer file example in Defining a printer file shows how to code the name type.

For more information about types of names, see Name for printer files (positions 19 through 28).
