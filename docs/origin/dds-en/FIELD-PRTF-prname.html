Name for printer files (positions 19 through 28)

Last Updated: 2026-07-13

You specify record format names and field names in these positions.

Name must begin in position 19.

The printer file example in Defining a printer file shows how to specify record format names and field names.

  • Record format name in printer files
    When you specify an R in position 17, the name specified in positions 19 through 28 is a record format name.
  • Field name in printer files
    When position 17 is left blank, the name specified in positions 19 through 28 is a field name.
  • Constant fields in printer files
    Constant fields are unnamed fields (positions 19 through 28 must be blank), or fields with a special name (*NONE) when the POSITION keyword is used in the record.
