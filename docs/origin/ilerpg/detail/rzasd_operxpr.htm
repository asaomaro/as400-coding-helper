拡張演算項目 2 の命令コード

Last Updated: 2021-04-13
有効な命令コード名を入力してください。 拡張演算項目 2 フィールドを使用す る命令コードには次のものがあります。
  • CALLP (プロトタイプ・プロシージャーまたはプログラムの呼び出し)
  • DOU (条件が真になるまでの繰り返し)
  • DOW (条件が真の間繰り返し)
  • ELSEIF (ELSE IF)
  • EVAL (式の評価)
  • EVALR (式の評価、右寄せ)
  • FOR (For)
  • IF (If)
  • ON-ERROR (エラーの時)
  • RETURN (呼び出し元への戻し)
  • WHEN (真の場合に選択)

拡張演算項目 2 フィールドを使用しない命令コードについては、命令コードを参照してください。
