Positions 7-16 (File Name)

Last Updated: 2026-07-13
Free-Form Syntax The name is specified following the DCL-F operation code. See Free-Form File Definition Statement
Entry
Explanation
A valid file name
Every file used in a program or procedure must have a unique name. The file name can be from 1 to 10 characters long, and must begin in position 7.

See Rules for File Names for more information about how the file name is used at compile-time and run-time.
