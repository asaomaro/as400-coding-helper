Positions 21-29 (External File Record ID Indicators)

Last Updated: 2026-07-13

Output indicators for externally described files are specified in the same way as those for program described files. The overflow indicators OA-OG, OV are not valid for externally described files. For more information on output indicators, see Positions 21-29 (File Record ID Indicators).
